# Agent Instructions

1. All documentation is in the `docs/` folder.
2. Do not delete files or run Git commands without user permission.
3. Always plan first before executing.
4. Always update `docs/changelog.md` after any changes. Use the following format:
   - **Date > File name > methods or functions > what the change does**
   - Each change should be on a new bullet point
5. Before installing dependencies or creating additional files, get user permission and explain why they are needed.
